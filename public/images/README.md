# Images

Drop your real visuals here, then set the matching `image:` path in
`lib/content.ts`. Until a path is set, an elegant placeholder renders.

Suggested files (any high-quality JPG/PNG/WebP works):

| Use | Set in `lib/content.ts` | Suggested file | Best ratio |
|-----|-------------------------|----------------|-----------|
| Hero visual | `hero.image` | `hero.jpg` | tall (4:5) |
| AI Photoshoots service | `services[0].image` | `service-photoshoots.jpg` | 4:3 |
| Instagram service | `services[1].image` | `service-instagram.jpg` | 4:3 |
| Ads service | `services[2].image` | `service-ads.jpg` | 4:3 |
| Marigold Miraaya case | `caseStudies[0].image` | `work-marigold.jpg` | 4:3 |
| Brand B case | `caseStudies[1].image` | `work-brand-b.jpg` | 4:3 |
| Brand C case | `caseStudies[2].image` | `work-brand-c.jpg` | 4:3 |

Example: after adding `hero.jpg`, set `hero.image = "/images/hero.jpg"`.
